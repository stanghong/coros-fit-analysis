Speed & Stroke Rate Trends

üìä How to Read This Chart

Blue bars = Speed (m/s)
Higher bars = faster swimming speed. Look for consistent bar heights = steady pace.

Orange bars = Stroke Rate (scaled)
Higher bars = more strokes per minute. Consistent = good technique.

Tips for improvement:

  • If speed drops but stroke rate stays high = fatigue management
  • If stroke rate drops = focus on stroke drills
  • If both drop together = build endurance
  • If inconsistent = practice pacing with interval training

Speed vs Efficiency Cloud

üìä How to Read This Chart

Efficiency = Speed / Stroke Rate
Higher efficiency = more distance per stroke (better technique).

Top-right = Best: High speed with high efficiency
Bottom-left = Needs Work: Low speed with low efficiency

Tips for improvement:

  • If points are bottom-left = focus on stroke technique
  • If points are top-left = increase stroke rate to gain speed
  • If points are bottom-right = maintain efficiency while building speed
  • If points are scattered = work on consistency
